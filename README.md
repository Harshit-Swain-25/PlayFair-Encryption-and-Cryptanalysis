# Playfair Cipher Implementation using Python

A Python-based implementation of the Playfair Cipher, a classical encryption technique that secures messages by encrypting pairs of letters instead of individual characters. This project also includes key matrix generation, comparative analysis, frequency visualization, and a simple cryptanalysis demonstration.

## 📌 Project Overview

This project demonstrates how the Playfair Cipher works by generating encryption matrices and applying encryption and decryption algorithms to plaintext messages.

The implementation supports both row-wise and column-wise key matrix generation methods and compares their outputs through digraph frequency analysis.

Additionally, the project includes a basic brute-force attack demonstration to illustrate cryptanalysis concepts.

This project is useful for understanding fundamental cryptography principles and classical encryption techniques.

## 🚀 Features

* Playfair Cipher encryption
* Playfair Cipher decryption
* Row-wise key matrix generation
* Column-wise key matrix generation
* Plaintext preprocessing
* Digraph creation
* Digraph frequency analysis
* Comparative analysis between encryption methods
* Hamming distance calculation

* Basic brute-force cryptanalysis demonstration

## 🔐 Project Workflow

1. Enter a secret key.
2. Generate a 5×5 Playfair key matrix.
3. Preprocess the plaintext.
4. Create character pairs (bigrams).
5. Encrypt the plaintext.
6. Decrypt the generated ciphertext.
7. Compare row-wise and column-wise encryption results.
8. Visualize digraph frequencies.
9. Perform a basic brute-force cryptanalysis demonstration.
